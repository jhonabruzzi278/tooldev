## ¿Qué es?

Footer.design es una galería especializada que recopila y cura diseños de pie de página (footers) de sitios web reales. Es un recurso de referencia enfocado exclusivamente en la sección del footer, permitiendo a diseñadores encontrar inspiración específica para este componente frecuentemente descuidado.

## ¿Para qué sirve?

Footer.design sirve para encontrar inspiración y referencias cuando necesitas diseñar o rediseñar el pie de página de un sitio web. Permite identificar patrones de diseño, estructuras de información y estilos visuales que funcionan bien en footers de diferentes industrias y tipos de sitio.

## Cuándo usarla

- Cuando estás diseñando el footer de un nuevo sitio web y necesitas referencias específicas.
- Para optimizar un footer existente que no cumple su función de navegación y conversión.
- Si quieres identificar patrones de footer comunes en tu industria o tipo de producto.
- Cuando necesitas mostrar al equipo ejemplos concretos de cómo otros resuelven el footer.

## Cuándo NO usarla

- Si buscas inspiración para secciones completas de un sitio (usa Land-book o Dribbble).
- Cuando necesitas templates de footer listos para implementar con código.
- Para diseñar headers, heroes u otras secciones que no son pie de página.
- Si ya tienes un design system definido y solo necesitas implementar el footer.

## Pros

- Recurso 100% gratuito y de libre acceso.
- Enfoque hiperespecializado en un componente que suele ser difícil de diseñar bien.
- Muestra footers reales de sitios web funcionales.
- Ahorra tiempo al no tener que buscar footers en galerías genéricas.
- Útil para justificar decisiones de diseño con ejemplos concretos.

## Contras

- Alcance limitado solo a footers (no cubre otras secciones del sitio).
- Puede tener menos actualizaciones que galerías más grandes y generales.
- No incluye código ni implementación técnica de los footers mostrados.
- La categorización puede ser menos robusta que plataformas más grandes.
- No ofrece métricas de efectividad o conversión de los diseños mostrados.
