## ¿Qué es?

HTMLrev es un directorio curado de más de 1500 plantillas HTML gratuitas. Cada plantilla es seleccionada manualmente por calidad y utilidad, cubriendo landing pages, portfolios, blogs, tiendas y dashboards.

## ¿Para qué sirve?

HTMLrev sirve para encontrar plantillas web gratuitas y de calidad para cualquier tipo de proyecto. Es útil para desarrolladores que necesitan una base rápida para sus sitios, diseñadores que buscan inspiración o equipos con presupuesto limitado.

## Cuándo usarla

- Cuando necesitas una landing page rápida para un proyecto.
- Para buscar inspiración de diseño web.
- Si trabajas con un framework específico (React, Vue, Tailwind) y quieres plantillas compatibles.
- Para proyectos con presupuesto limitado que requieren diseño profesional.

## Cuándo NO usarla

- Si necesitas soporte técnico o personalización incluida (mejor contratar un diseñador).
- Para proyectos que requieren diseño 100% original y único.
- Si buscas templates con backend integrado (son solo frontend).
