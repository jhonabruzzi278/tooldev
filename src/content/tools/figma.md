## ¿Qué es?

Figma es una herramienta de diseño de interfaces basada en la nube que permite crear, prototipar y colaborar en diseños de productos digitales. Funciona directamente en el navegador, lo que elimina la necesidad de instalar software y permite que diseñadores y desarrolladores trabajen en el mismo archivo simultáneamente.

## ¿Para qué sirve?

Figma sirve para diseñar interfaces de usuario (UI) y experiencias de usuario (UX), crear prototipos interactivos para validar ideas, construir y mantener sistemas de diseño escalables, colaborar en tiempo real con equipos multidisciplinarios y entregar especificaciones de diseño a desarrolladores.

## Cuándo usarla

- Para diseñar interfaces web y móviles de forma profesional.
- Cuando tu equipo necesita colaborar en tiempo real sobre los mismos diseños.
- Para crear prototipos interactivos y validar flujos de usuario.
- Si necesitas un sistema de diseño centralizado y escalable.
- Cuando desarrolladores necesitan inspeccionar diseños para implementar CSS.

## Cuándo NO usarla

- Si necesitas una herramienta completamente open source y autogestionada.
- Para edición de imágenes fotográficas o ilustraciones complejas (usa Photoshop o Illustrator).
- Cuando trabajas sin conexión a internet de forma frecuente.
- Si tu presupuesto es limitado y necesitas funcionalidades avanzadas del plan de pago.

## Pros

- Colaboración en tiempo real líder en la industria.
- Funciona en cualquier sistema operativo desde el navegador.
- Curva de aprendizaje suave para principiantes.
- Dev Mode facilita la comunicación entre diseño y desarrollo.
- Ecosistema de plugins y comunidad muy activos.

## Contras

- No es open source y depende de los servidores de Figma.
- Los planes de pago pueden ser costosos para equipos grandes.
- Requiere conexión a internet para la mayoría de funciones.
- Archivos muy grandes pueden tener problemas de rendimiento.
- Exportación de código limitada comparada con herramientas dedicadas.
