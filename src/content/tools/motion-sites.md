## ¿Qué es?

Motion Sites es una colección curada de sitios web que destacan por la calidad excepcional de sus animaciones y motion design. Funciona como una galería de referencia donde desarrolladores y diseñadores pueden descubrir implementaciones innovadoras de movimiento en la web, desde micro-interacciones hasta animaciones de scroll complejas.

A diferencia de directorios generales como Awwwards, Motion Sites se especializa exclusivamente en el aspecto de animación y movimiento, filtrando solo aquellos sitios donde la calidad del motion design es verdaderamente sobresaliente.

## ¿Para qué sirve?

Motion Sites sirve como fuente de inspiración y referencia visual para cualquier proyecto que necesite animaciones de calidad. Permite descubrir tendencias actuales en motion design web, encontrar referencias para presentaciones a clientes, y estudiar implementaciones reales de efectos que quieras replicar en tus propios proyectos. Es un recurso de aprendizaje continuo sobre lo que es posible lograr con animación web moderna.

## Cuándo usarla

- Cuando inicias un proyecto y necesitas definir el nivel y estilo de animación deseado.
- Para crear mood boards y colecciones de referencia visual para equipos de diseño.
- Cuando quieres mantenerte actualizado sobre tendencias en motion design web.
- Para presentar ejemplos concretos de animación a clientes o stakeholders.
- Cuando buscas inspiración técnica para implementar un efecto específico.

## Cuándo NO usarla

- Cuando necesitas implementaciones de código listas para copiar (usa React Bits o CodePen).
- Para aprender los fundamentos básicos de CSS animations o JavaScript (usa MDN o cursos).
- Si buscas templates o boilerplates completos con animaciones incluidas.
- Cuando necesitas documentación técnica detallada sobre una librería de animación específica.

## Pros

- Completamente gratuito sin necesidad de registro.
- Curaduría enfocada exclusivamente en calidad de motion design.
- Actualización regular con nuevos sitios inspiradores.
- Perfecto como herramienta de comunicación con clientes y equipos.
- Sirve como termómetro de tendencias actuales en animación web.

## Contras

- No proporciona código ni implementaciones técnicas directamente.
- Requiere trabajo adicional de investigación para replicar los efectos mostrados.
- La colección puede estar sesgada hacia sitios de agencias y portfolios creativos.
- No incluye análisis de rendimiento o accesibilidad de las animaciones mostradas.
- Algunos sitios referenciados pueden dejar de estar disponibles con el tiempo.
