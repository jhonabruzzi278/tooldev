## ¿Qué es?

Framer es una herramienta de diseño que combina prototipado interactivo con publicación web real. A diferencia de Figma (solo diseño), Framer permite crear prototipos 100% interactivos con animaciones, lógica y publicarlos como sitios web funcionales.

## ¿Para qué sirve?

Framer sirve para diseñar prototipos de alta fidelidad con animaciones reales, crear landing pages y sitios web sin código, presentar diseños interactivos a clientes, y publicar sitios con CMS integrado.

## Cuándo usarla

- Cuando necesitas prototipos interactivos con animaciones reales.
- Para diseñar y publicar landing pages sin escribir código.
- Si buscas una alternativa a Webflow con mejores capacidades de diseño.
- Cuando presentas prototipos a clientes que pueden interactuar.
- Para equipos pequeños que diseñan y publican directamente.

## Cuándo NO usarla

- Si eres diseñador UI/UX profesional centrado en diseño de sistemas (Figma es mejor).
- Para proyectos que requieren prototipado complejo de apps nativas.
- Cuando necesitas un CMS enterprise con miles de páginas.
- Si tu equipo ya está establecido en Figma con librerías compartidas.

## Pros

- Prototipos con animaciones reales sin código.
- Publicación web directa desde el diseño.
- CMS integrado para contenido dinámico.
- Componentes con variantes y overrides.
- Ideal para freelancers y agencias pequeñas.

## Contras

- Menos funcionalidades de diseño colaborativo que Figma.
- Plan gratuito muy limitado.
- No es ideal para sistemas de diseño grandes.
- Menos plugins y comunidad que Figma.
- El plan Pro es costoso para uso individual.
